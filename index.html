Murrman's Home

Hello and welcome! I'm Conor Murray. I am a:

  • - Fullstack Software Engineer
  • - Film Maker
  • - Photographer
  • - Cat Dad

I've just launched this site so there's a lot I want to add over time. For now I can link a few films I'm working on with talented folks and dear friends (not mutually exclusive), some hobby sites I work on when I can, photography (eventually, when I get back to it, hopefully some astro stuff), ways to talk to me and HUNDREDS OF PHOTOS OF MY CAT.

I always liked discovering some random person's website that they'd made for themselves. There's something very wholesome about it; people just putting their little world out there. This is my go at it.

I want this to be a kind of playground. Already I'm using the Speculation Rules API to use and abuse your bandwidth. If or when I use Javascript it's going to be import maps all the way! Nothing against bundlers or the current eco-system here. Find plenty of that if you search for it. I just want this site to help me stretch my legs in other areas a bit.
